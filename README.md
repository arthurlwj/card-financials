

# Card Financials API

API NestJS + TypeORM + Postgres

## Description

Documentação referente ao projeto de gestão de gastos

## Rodando local
```bash
cp .env.example .env
npm i
npm run start:dev

# Swagger: http://localhost:3000/api


## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
